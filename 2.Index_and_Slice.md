# Index and Slice索引与切片

## 创建一个tensor

``` Python
import torch

a = torch.rand(4,3,28,28)
# 4张照片 3个通道 大小为28*28
a.shape
#查看a的形状
```

## 1.使用":"进行切片

```Python
a[:2,:1,:,:].shape

a[:,:,0:28:2,0:28:2].shape# 进行跳跃切片
```

## 2.select by specifuc index

```Python
a.index_select(0,torch.tensor([0,2])).shape
# 表示对第0维，取0和2两张图片
# 注意第二个参数要使用tensor数据类型
```

## 3. 使用"..."简化代码

```Python
a[0,0,...].shape
# "..."表示取后面维度全部的数据
```
